5.配置Github仓库

  • 创建仓库，名称为yourname.github.io。

  • 在git bash中配置github账户信息。

  • 创建本机密钥

  • 将ssh公钥存入github中：（点击头像中的setting选项，选择ssh-key，新建ssh-key，key复制id_rsa.pub中的内容）

8.配置主题

  • 默认主题是landscape，其配置文件在theme文件夹中，如需更改主题可将目标主题文件存入theme中，并在_config.yml _中更改主题名为目标主题。如该页面主题为cafe，可在github中下载后，重命名为cafe，再存入theme中。更改_config.yml _中的theme为cafe。

  • 生成静态文件并部署

  • 在github仓库中存储的文件是本地source文件夹和theme文件夹里的使用主题下的内容，博客配置文件并没有被上传到github中。

安装python2和python3

1.Windows下

1.1、安装python2和python3

a.链接：

  • python3.5.2
  • python2.7.9

b.配置环境变量

  • （假定python2安装目录为D:\Python\python27,配置python3类似）
  • D:\Python\python27
  • D:\Python\python27\Scripts

c.找到python3 IDLE安装位置，发送快捷方式到桌面

1.2、修改python.exe名字

  • 修改python2.7.9和python3.5.2安装目录中的 python.exe和pythonw.exe的名称为python2.exe、pythonw2.exe和python3.exe、pythonw3.exe。
  • （cmd命令，输入python2即可运行python2.7.9版本，python3即可运行python3.5.2鞍姹荆?

1.3、pip2 和 pip3设置

python安装包需要用到包管理工具pip，由pip安装的包的位置（根据个人安装）目录如下：D:\Python\Python37\Lib\site-packages

  • 在DOS命令框输入命令，python3 -m pip install –upgrade pip –force-reinstall，显示重新安装成功。

  • 在DOS命令框输入命令，python2 -m pip install –upgrade pip –force-reinstall，显示重新安装成功。【1】
